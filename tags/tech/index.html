Runalyze - Laufstatistiken soweit man Laufen kann

June 3, 2015

Ich bin vor ca. 3 Tagen auf ein mir bis dato unbekanntes Tool Namens Runalyze gestoßen. Zugegeben der Titel es Beitrages trifft nicht auf jeden zu, einige können sicherlich deutlich weiter Laufen als es Statistiken gibt. Das Tool möchte ich hier mal kurz vorstellen, einige kennen es vielleicht schon. Vielleicht kann auch nicht jeder etwas damit anfangen. Wenn man allerdings auf Statistiken zum Laufen, so wie ich, steht dann ist dieses Tool eine passende Ergänzung.
